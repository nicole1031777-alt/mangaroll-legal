## ❓ 常见问题 · FAQ

### 生成失败,钻石被扣了怎么办?
作品未能成功生成时,系统会**自动退还**钻石或 VIP 次数。如果未自动退还,请联系上方邮箱并附上 UID。

### 怎么删除我的作品?
- 灵感广场:进入帖子详情 → 右上 「⋯」 → 删除
- 创作历史:长按缩略图 → 选择「删除」
- 注销账号:个人中心 → 设置 → 注销账号(永久删除全部作品 + 不可恢复)

### 如何举报不当内容?
任何作品详情页右上 「⋯」 → 「举报」,选举报理由提交,我们会在 24 小时内人工审核处理。

### 订阅怎么取消?
- iPhone 设置 → Apple ID → 订阅 → MangaRoll → 取消订阅
- 订阅会在到期前 24 小时自动续费,取消后已支付期间仍可使用至到期

### 我能拉黑其他用户吗?
- 在任何帖子或评论旁点该用户头像 → 进个人主页 → 右上「⋯」→ 「拉黑」
- 拉黑后:对方的帖子 / 评论 / 私信 都不会再显示
- 管理已拉黑列表:个人中心 → 设置 → 已拉黑用户
